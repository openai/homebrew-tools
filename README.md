# OpenAI Homebrew Tools

Homebrew tap for OpenAI command-line tools.

## OpenAI CLI

This tap publishes the `openai` cask for the OpenAI CLI once public release artifacts are available.

```sh
brew install openai/tools/openai
```

The cask definition is generated by the OpenAI CLI release workflow.

## Tart projects

The Tart release workflows publish the following formulas through GoReleaser:

```sh
brew install openai/tools/tart
brew install openai/tools/softnet
brew install openai/tools/orchard
brew install openai/tools/tart-guest-agent
```

Each release opens a single-recipe pull request. The tap validates the generated
recipe and auto-merges updates authored by the `openai-homebrew-releaser` App.
